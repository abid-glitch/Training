Abbreviations

This element is used to mark up acronyms or short forms:

cdgl started in 2020.

Blockquote Example

This tag is used to quote content from other sources:

Codingal has a vision to inspire every school child

Source: Codingal

Code with Multiple Lines

To display multiple lines of code, use the pre tag:

        codingal works on
        a mission to
        inspire school
        children

Keyboard Entry

Use the kbd tag to show keyboard shortcuts or input:

Press ctrl + p to bring up the print menu.
